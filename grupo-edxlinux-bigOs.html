Instalação de Componentes Essenciais

Para uma experiência completa, certifique-se de que os seguintes pacotes estejam instalados:

  • LXDM: Um gerenciador de exibição leve.
  • Ubiquity Frontend GTK: A interface gráfica do instalador Ubiquity.
  • Ubiquity Slideshow Lubuntu: O slideshow de instalação específico para Lubuntu.

Comandos de Instalação no Terminal:

sudo apt install lxdm ubiquity-frontend-gtk ubiquity-slideshow-lubuntu

Ou, você pode instalar o Lubuntu completo via Synaptic, que geralmente inclui a maioria desses componentes.

---

Configuração do Plymouth

O Plymouth é o sistema que exibe a tela de boot gráfica. Para garantir que o logo do Lubuntu seja exibido corretamente, um arquivo específico precisa ter permissões de execução.

Tornando o Arquivo Plymouth Executável:

O arquivo /usr/share/plymouth/themes/lubuntu-logo/lubuntu_logo.png, apesar de ser uma imagem, às vezes precisa de permissão de execução para certas configurações do Plymouth. Você pode fazer isso de duas maneiras:

Método 1: Comando Direto

sudo chmod +x /usr/share/plymouth/themes/lubuntu-logo/lubuntu_logo.png

Método 2: Navegando até o Diretório

Primeiro, vá até o sistema com o comando:

cd /usr/share/plymouth/themes/lubuntu-logo

E depois use o comando:

sudo chmod +x lubuntu_logo.png

Instalando o Pacote Lubuntu-BigOS (Exemplo)

Se você tem um pacote chamado lubuntu-bigOs.tar.gz, siga os passos abaixo para copiá-lo para o sistema e descompactá-lo:

Passos para Instalação:

  1. Baixe o pacote: Certifique-se de que lubuntu-bigOs.tar.gz esteja no seu diretório atual (ou ajuste o caminho).
  2. Copie para o sistema: Copie o arquivo para /usr/share.
    3. sudo cp -r lubuntu-bigOs.tar.gz /usr/share
  3. Descompacte: Navegue até o diretório e descompacte o arquivo.
    4. cd /usr/share
sudo tar -vzxf lubuntu-bigOs.tar.gz
  4. Remova o arquivo compactado: Após a descompactação, você pode remover o arquivo .tar.gz para economizar espaço.
    5. sudo rm lubuntu-bigOs.tar.gz

Lubuntu BigOS - Instruções Adicionais

Este guia fornece instruções detalhadas para a instalação de componentes, configuração do Plymouth e gerenciamento de arquivos relacionados ao projeto Lubuntu BigOS.

---

1. Instalação de Pacotes Essenciais

Para garantir que o sistema tenha todos os componentes necessários, você pode instalar os pacotes abaixo usando o terminal ou o Synaptic.

Via Terminal:

sudo apt install lxdm ubiquity-frontend-gtk ubiquity-slideshow-lubuntu

Via Synaptic:

Se preferir uma interface gráfica, você pode instalar o Lubuntu completo ou procurar por esses pacotes individualmente no gerenciador de pacotes Synaptic.

---

2. Configuração do Plymouth

O Plymouth é responsável pela tela de inicialização. Para que o tema seja exibido corretamente, é necessário garantir que o arquivo de imagem do logo tenha permissão de execução.

Torne o arquivo executável com chmod:

sudo chmod +x /usr/share/plymouth/themes/lubuntu-logo/lubuntu_logo.png

Você também pode navegar até o diretório e executar o comando lá:

cd /usr/share/plymouth/themes/lubuntu-logo
sudo chmod +x lubuntu_logo.png

Nota: Embora seja um arquivo de imagem, a permissão de execução pode ser necessária para que o Plymouth o processe corretamente como parte do tema.

---

3. Instalação do Pacote lubuntu-bigOs

Siga estes passos para copiar e descompactar o pacote do projeto em seu sistema:

  • Baixar o arquivo: Certifique-se de que o arquivo lubuntu-bigOs.tar.gz esteja no seu sistema.
  • Copiar para o sistema:
    • sudo cp -r lubuntu-bigOs.tar.gz /usr/share
  • Descompactar o arquivo:
    • cd /usr/share
sudo tar -vzxf lubuntu-bigOs.tar.gz
  • Remover o arquivo original:
    • sudo rm lubuntu-bigOs.tar.gz
---

4. Atualizando o initramfs

Para que as alterações no tema do Plymouth entrem em vigor, é crucial atualizar o initramfs.

sudo update-initramfs -u

Este comando reconstrói a imagem inicial do sistema de arquivos, garantindo que as novas configurações de boot sejam aplicadas na próxima inicialização.
